# Library Book Request System in ServiceNow

## Overview

The **Library Book Request System** is a ServiceNow-based solution designed to simplify the process of requesting books from a library. Students can submit book requests through a self-service catalog, while librarians can review, approve, issue, or reject requests efficiently. The system automates the complete request lifecycle, improving library operations and user experience.

## Project Objectives

- Automate library book request submission
- Reduce manual paperwork and approval delays
- Improve book issue tracking
- Provide request status visibility
- Enhance library management efficiency

## Key Features

- Self-Service Book Request Catalog
- Student Information Capture
- Book Selection
- Automated Request Number Generation
- Librarian Approval Workflow
- Issue & Return Tracking
- Email Notifications
- Real-Time Status Updates
- Reporting & Dashboard Support
